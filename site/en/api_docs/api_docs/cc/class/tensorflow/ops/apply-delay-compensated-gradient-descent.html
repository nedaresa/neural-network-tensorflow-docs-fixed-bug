tensorflow::ops::ApplyDelayCompensatedGradientDescent

#include <training_ops.h>

var -= alpha * (delta + lambda * delta * (var - shadow))

Summary

Update '*shadow' by changing it to the new value of 'var'

Arguments:

  • scope: A Scope object
  • var: Should be from a Variable().
  • alpha: Scaling factor. Must be a scalar.
  • delta: The change.
  • lambda: The variance parameter.
  • shadow: Same as "var".

Optional attributes (see Attrs):

  • use_locking: If True, the subtraction will be protected by a lock; otherwise the behavior is undefined, but may exhibit less contention.

Returns:

  • the created Operation
