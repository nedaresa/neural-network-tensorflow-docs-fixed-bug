Install TensorFlow with Python's pip package manager. TensorFlow is supported and tested on the following 64-bit systems:

  • Ubuntu 16.04 or later
  • macOS 10.12.6 (Sierra) or later (no GPU support)
  • Windows 7 or later
  • Raspbian 9.0 or later
Note: Starting with TensorFlow 1.6, binaries use AVX instructions which may not run on older CPUs.

Available packages

Choose one of the following TensorFlow packages to install:

  • tensorflow —Current release for CPU-only (recommended for beginners)
  • tf-nightly —Nightly build for CPU (unstable)

Programs using the TensorFlow GPU package usually run much faster. Read the GPU support guide to set up a CUDA®-enabled GPU card on Ubuntu or Windows.

  • tensorflow-gpu —Current release with GPU support
  • tf-nightly-gpu —Nightly build with GPU support (unstable)

2. Create a virtual environment (recommended)

Virtual environments are used to isolate packages from the system. Create a new virtual environment by choosing a Python interpreter and making a ./venv directory to hold it:

{% dynamic if request.query_string.lang == "python2" %} 
virtualenv --system-site-packages -p python2.7 ./venv
{% dynamic else %} 
virtualenv --system-site-packages -p python3 ./venv
{% dynamic endif %}

Activate the virtual environment using a shell-specific command:

Sh

source ./venv/bin/activate  # sh, bash, ksh, or zsh

Csh

source ./venv/bin/activate.csh  # csh or tcsh

Fish

. ./venv/bin/activate.fish

When virtualenv is active, your shell prompt is prefixed with (venv).

Install packages within a virtual environment without affecting the host system setup. Start by upgrading pip: 

pip install --upgrade pip

pip list  # show packages installed within the virtual environment

And to exit virtualenv later:

deactivate  # don't exit until you're done using TensorFlow

3. Install the TensorFlow pip package

Choose one of the following TensorFlow packages to install from PyPI:

  • tensorflow —Current release for CPU-only
  • tf-nightly —Nightly build for CPU (unstable)
  • tensorflow-gpu —Current release with GPU support (With a CUDA®-enabled GPU card on Ubuntu or Windows.)
  • tf-nightly-gpu —Nightly build with GPU support (unstable).
Package dependencies are automatically installed. They're listed in the setup.py file under REQUIRED_PACKAGES.

Virtualenv install

pip install --upgrade tensorflow

Verify the install:

python -c "import tensorflow as tf; print(tf.__version__)"

System install

{% dynamic if request.query_string.lang == "python2" %} 
pip install --user --upgrade tensorflow  # install in $HOME

Verify the install:

python -c "import tensorflow as tf; print(tf.__version__)"
{% dynamic else %} 
pip3 install --user --upgrade tensorflow  # install in $HOME

Verify the install:

python3 -c "import tensorflow as tf; print(tf.__version__)"
{% dynamic endif %}
Success: TensorFlow is now installed. Read the tutorials to get started.
