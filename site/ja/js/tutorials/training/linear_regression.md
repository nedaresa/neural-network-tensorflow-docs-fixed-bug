# 2次元データを予測する

このチュートリアルでは、一連の自動車を表現した数値データを予測するモデルを訓練します。

この演習は様々な種類のモデルの訓練に共通する手順を紹介するものですが、使用するのは小さなデータセットと単純な（浅い）モデルです。主な目的はTensorFlow.jsでモデルを訓練する際に使用する基本的な用語や概念、構文に慣れる手助けをし、さらに調査と学習を進める足がかりを提供することです。
